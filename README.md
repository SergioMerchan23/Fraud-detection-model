# Fraud-detection-model

Descripcion: Desarrolle un modelo predictivo de fraudes en transacciones en un e-commerce, puede ser usado como base el servicio de seguridad de cualquier e-commerce. Las metricas de evaluacion que se utilizo para ver que tan bueno era el modelo fue la Precision, Recall, el F1 y tambien AUC_ROC.

Tecnoligia y librerias utilizadas: Utilice Python y las librerias pandas, sklearn, matplotlib, numpy y seaborn. Ademas utilice el modelo RandomForest y el modelo XGBoost

Responsabilidad clave: Ademas de la limpieza de datos, se realizo una EDA (analisis exploratorio de datos), se crearon dos modelos, un random forest que nos dio metricas de evaluacion de base para compararlas con el modelo XGBoost que seria el de mejor resultados.

Impacto logrado: Se obtuvo un modelo con Precision: 85%, Recall: 82%, F1: 83% y un AUC-ROC:0.9917, que nos demuestra que el modelo es muy robusto y eficiente.
